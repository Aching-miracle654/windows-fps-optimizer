## ❓ Troubleshooting

Most users find the tool works without issue. If you encounter unexpected behavior, try these steps.

**The installation stops midway.**
Ensure you have an active internet connection. Check that your antivirus is not blocking the installation. You can temporarily disable the protection during the setup process.

**The program does not launch.**
Right-click the application icon and select Run as Administrator. This gives the software the permission it needs to modify system settings.

**The computer feels slow after optimization.**
This is rare. If it happens, open the software and click the Restore Defaults button. This clears all active changes and returns your system to the settings Windows chose at setup.

**The software alerts that it cannot find the game.**
The software automatically detects popular Steam and Epic Games launchers. If you installed your game in a non-standard folder, click the Add Game button and select the folder where the game stores its main executable file.
